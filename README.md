# PinSetterApp
JavaFX PIN Setter app that allows users to create a password with confirmation and validity.


FEATURES
Only allowed to input a PIN that contains 4 digits.
Requires user to confirm PIN after first input.
Weak PINs are not allowed (No repeated digits, no sequential numbers).
Displays to show if PIN was sucessful or not.
Visibally changes color based on PIN validity.

HOW IT WORKS
Open the project in IntelliJ IDEA (IDE).
Run the HelloApplication class.
Enter a 4-digit PIN.
Confirm the PIN.
Click "Set PIN".
